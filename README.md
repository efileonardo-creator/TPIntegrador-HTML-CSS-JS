Consigna del trabajo final:


Crear un formulario que implemente validaciones de datos utilizando JavaScript.

También el formulario debe poseer una sección donde se muestren los mensajes de las validaciones.

Implementar validaciones de formato , por ejemplo en los campos de texto, numéricos , email utilizando JS. 
En caso de que los datos sean correctos, redireccionar a otra página
O mostrar en la sección de mensajes "Envío exitoso".

Secciones que pueden crear: 
Ej: Formulario de login con un campo para Usuario/Email otro para la contraseña y un botón que permita el llamado a la validación con JS.
Los tipos de campos y formulario son personalizados a gusto personal.
